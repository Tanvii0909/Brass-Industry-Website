# Brass Industry Website
## Overview
Brass Industry Website is a PHP-based web application developed as an academic project.
It provides an online presence for a brass industry by showcasing information, products, and services in a structured manner.

## Academic Details
- Course : BCA
- Semester : 6
- Project Type: Academic Project

## Technologies Used

- Backend : PHP
- Database : MySQL
